📊 Sales Dashboard Analysis using Power BI
Project Overview
Developed an interactive sales dashboard in Power BI to analyze sales performance, customer demographics, and order trends across different channels and cities.

Tools Used
* Power BI
* Excel
* DAX
* Power Query

Dashboard KPIs
* Total Sales: 21M
* Total Orders: 31K
* Average Order Value: 682

Analysis Performed
* Monthly sales growth analysis
* Top 5 cities with highest orders
* Order status analysis (Delivered, Returned, Cancelled, Refunded)
* Sales by product category
* Channel-wise sales distribution
* Gender-wise sales analysis
* Age group analysis

Key Insights
* Bengaluru generated the highest number of orders.
* Amazon contributed the largest share of sales.
* Female customers accounted for the majority of sales.
* Adults represented the highest revenue-generating age group.
* Most orders were successfully delivered.

Skills Demonstrated
* Data Cleaning using Power Query
* Data Modeling
* DAX Measures and Calculations
* Interactive Dashboard Design
* Business Intelligence and Data Visualization
